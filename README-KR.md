PlemolJP는 IBM Plex Mono 시리즈를 기반으로 하며, 노멀과 이탤릭 스타일을 지원합니다. 각 스타일마다 8가지 웨이트(Thin~Bold)를 제공합니다.

또한 일본어 환경에서 프로그래밍 시 자주 발생하는 전각 스페이스 오입력을 인지할 수 있도록 전각 스페이스를 가시화했습니다.

💡 전각 스페이스 가시화가 필요 없는 경우, 릴리스의 Assets에서 `PlemolJP_HS_vx.x.x.zip` 형식의 zip 파일을 선택하세요. (HS: Hidden Space)

💡 Powerline 기호 등이 포함된 Nerd Fonts 대응 버전은, 릴리스의 Assets에서 `PlemolJP_NF_vx.x.x.zip` 형식의 zip 파일을 선택하세요. (NF: Nerd Fonts)

| 폰트 패밀리 | 설명 |
|------------|------|
| **PlemolJP** | 문자 폭 비율이 「반각 1:전각 2」인 일반 버전. 주로 ASCII 코드의 영숫자 기호에 IBM Plex Mono의 자체를 사용하고, 그 외의 일본어 문자나 기호류에는 IBM Plex Sans JP를 사용합니다. |
| **PlemolJP Console** | IBM Plex Mono의 자체를 제외하지 않고 모두 적용한 폰트 패밀리. 화살표 기호 등 많은 기호가 반각으로 표시되므로, 콘솔에서의 이용이나 기호류를 가능한 한 반각으로 표시하고 싶은 분에게 추천합니다. |
| **PlemolJP35** | 일반 버전 PlemolJP의 문자 폭 비율을 「반각 3:전각 5」로 한 폰트 패밀리. 영숫자가 일반 버전 PlemolJP보다 크게 표시됩니다. 일본어가 적은 문서나 코드의 경우에는 이쪽이 읽기 쉬울 수 있습니다. |
| **PlemolJP35 Console** | PlemolJP Console의 문자 폭 비율을 반각 3:전각 5로 한 폰트 패밀리 |
